# project-summary.md

## الهدف:
منصة سوق B2B مخصصة لقطاع المعارض والمؤتمرات والفعاليات، تجمع بين العملاء والموردين مع نظام إدارة الطلبات وعروض الأسعار والدفع الإلكتروني.

## الكيانات الرئيسية:
- المستخدم (عميل/مورد/مدير)
- الخدمات الرئيسية والتفصيلية
- الطلبات والعروض
- المدفوعات والإعلانات والدعم الفني

## البنية التقنية:
- Next.js + React + Tailwind + Prisma + PostgreSQL
- Tap Payments API
- REST APIs بتقسيم واضح لكل وحدة
- Modular SaaS Codebase

## تدفق العمل:
1. تسجيل عميل/مورد
2. إنشاء/مطابقة الطلبات
3. إرسال وقبول عروض الأسعار
4. دفع العمولة (5%) عبر Tap
5. أرشفة ومتابعة حالة الطلب
6. إدارة كاملة عبر لوحة المدير

## الإيرادات:
- عمولة ثابتة 5% من قيمة كل معاملة
- خيارات إعلانات كما خدمات مستقبلية

## قوائم التحقق:
- جميع المسارات الأساسية تم تخطيطها وتجهيزها للبرمجة فورًا
- كل صفحة وFeature مذكورة في خطة العميل موجودة فعليًا في الملفات
